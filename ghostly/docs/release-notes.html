1.6.0 Latest

New features:

  • Added link to review account permissions.

Bug fixes:

  • Notifications marked as read in GitHub was sometimes not marked as read in Ghostly.


1.5.0

New features:

  • Added ARM32 support. Ghostly now runs better on your Windows 10 tablet!

Bug fixes:

  • Fixed a query parsing bug that made it impossible to search for a single word without quoting it


1.4.0

Bug fixes:

  • Various bug fixes and stability improvements.


1.3.3

New features:

  • Auto completion for Ghostly's query language. This should make it easier to create filters, rules, and write queries.

Bug fixes:

  • Fixed issue where synchronization would take longer than it had to.
  • Various bug fixes and stability improvements.


1.2.1

New features:

  • Categories can now have custom icons. 🎉
  • The master/detail view can now be resized.

Bug fixes:

  • Fixed a lot of synchronization issues.
  • Changes from background synchronization should now update the UI if Ghostly is running.
  • Archiving all items in a category should make them as read as well.
  • Archive all should only marked unread items as read.
  • Remove delay when showing or hiding category badges.
  • Some commands were executed on the UI thread which would make Ghostly unresponsive.
  • Fixed issue where navigation list sometimes had an incorrect palette in dark mode.
